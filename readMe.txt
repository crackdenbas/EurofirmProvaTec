## Instalación
Instalación segun la documentación de React Native
https://reactnative.dev/docs/environment-setup

## Instalación de dependencias
npm install

## Ejecución
Necesitas tener instalado expo-cli para ejecutar expo-go en tu dispositivo móvil o emulador.
yarn android || yarn ios || expo start


## Compilar
yarn build:android || yarn build:ios || expo build:android || expo build:ios
